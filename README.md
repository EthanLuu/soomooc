# SooMooc 在线教学直播平台

该项目为『综合项目课程实践』的作业，仅供参考。

## 技术栈

- React 17.0.2
- React-Router 5.2.0
- Typescript
- Ant Design 
- flv.js 1.5.0
- json-server

## 启动项目

```shell
# 安装依赖
yarn 				# npm install

# 运行 json-server
yarn mock		# npm run mock

# 启动项目
yarn start 	# npm run start
```
